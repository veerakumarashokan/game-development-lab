# Base Concept

## Idea

A central base that acts as the main hub of the game.

## Features

- Player returns here after exploration  
- Serves as a safe and functional area  
- Can evolve into different types:
  - Laboratory  
  - Sanctuary  

## Design Approach

The base is kept flexible to support different gameplay directions and player roles in the future.

## Goal

To design a core hub that supports progression and system expansion.
