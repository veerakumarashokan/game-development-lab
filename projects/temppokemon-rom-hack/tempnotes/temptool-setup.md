# Tool Setup

## Tools Required

### Emulator
- VBA-M / mGBA (for testing ROMs)

### Legacy Tools
- Advance Map (map editing)
- Advance Text (text editing)

### Hybrid Tools
- Hex Maniac Advanced (advanced editing and scripting)

### Decomp Tools
- pokeemerald (source-based development)

---

## Purpose

These tools are used across different levels of ROM hacking:

- Legacy tools → basic editing (maps, text)  
- Hybrid tools → extended ROM modifications  
- Decomp tools → full custom game development  

---

## Workflow

- Edit maps using Advance Map  
- Modify data/scripts using Hex Maniac Advanced  
- Test changes using emulator  
- Use decomp tools for advanced systems later
