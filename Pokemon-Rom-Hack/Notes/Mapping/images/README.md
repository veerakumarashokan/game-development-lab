# Images

This folder contains visual references for mapping concepts and implementations.

## Naming Convention

Images follow a structured naming system:

<topic>-<stage><variant>.png

### Stages

- 0 → Original / unmodified state  
- 1 → First modification  
- 2 → Further improvements  

### Variants

- a, b → Minor variations within the same stage  

## Examples

- movement-permissions-0.png  
- movement-permissions-1.png  
- warp-setup-1a.png  
- warp-setup-2.png  

## Purpose

- Track visual progression of changes  
- Support documentation with clear references  
- Maintain organized and scalable image management
