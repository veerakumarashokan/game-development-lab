# Base Layout

## Structure

The base will be designed as a central hub with simple functional areas.

## Areas

- Entrance / Exit point  
- Main open area (central space)  
- Reserved space for future features  

## Design Approach

- Keep layout simple and expandable  
- Avoid complex design in early stage  
- Focus on functionality over detail  

## Goal

To create a basic map layout that can be expanded later.
