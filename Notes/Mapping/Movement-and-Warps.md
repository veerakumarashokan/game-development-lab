# Movement and Warp System

## Movement Permissions

Movement permissions define where the player can walk.

- Walkable tiles allow movement  
- Blocked tiles restrict access  
- Incorrect setup can cause glitches  

## Warp System

Warps connect different maps such as:

- Doors  
- Stairs  
- Transitions between locations  

## Key Points

- Entry and exit positions must align  
- Incorrect warp setup can break navigation  
- Must test using emulator  

## Testing

- Verified player movement

- Ensured no soft locks
